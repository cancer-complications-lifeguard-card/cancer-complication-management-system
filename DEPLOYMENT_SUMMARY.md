## 🏗️ 技术架构

### 前端架构
- **框架**: Next.js 15 (App Router)
- **UI库**: Tailwind CSS V3 + shadcn/ui组件
- **状态管理**: React Hooks + SWR
- **图表可视化**: Recharts + D3.js
- **地图服务**: Mapbox GL JS

### 后端架构
- **API路由**: Next.js API Routes
- **数据库**: SQLite (开发) / PostgreSQL (生产推荐)
- **认证**: JWT + bcrypt
- **文件上传**: 多媒体支持
- **实时通信**: WebSocket

### 安全架构
- **三重防护**: 数据加密 + 访问控制 + 审计日志
- **隐私保护**: 数据脱敏 + 权限分级 + 合规检查
- **医疗标准**: 符合医疗AI应用质量评价要求
